# Alephium frontend

A monorepo containing all things frontend on Alephium.

## Development

We use [Bun](https://bun.sh/) as our package manager and [Turborepo](https://turbo.build/repo) as our build system.

### Lint

```
bun lint
bun lint:fix
```

### Format

```
bun format
```

### Compile TypeScript

```
bun ts:check
```

### Build

```
bun run build
```
